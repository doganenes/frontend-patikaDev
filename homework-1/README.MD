# Ödev 1

## 1. Insertion Sort

- [22, 27, 16, 2, 18, 6]

- [22, 27, 16, 2, 18, 6] ... 1
- [16, 22, 27, 2, 18, 6]
- [2, 16, 22, 27, 18, 6]
- [2, 16, 18, 22, 27, 6] ... n-1
- [2, 6, 16, 18, 22, 27] ... n

## 2. Big-0 Notation

- worst case O(n^2)
- 0 + 1 + 2 + 3 + 4…..+ (n-1) = [n * (n-1)] / 2 : n^2

## 3. Time Complexity

- best case: O(n)
- average case: O(n^2)
- worst case: O(n^2)
